## 🚀 Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/adamwetzli/master_thesis
```

### 2. Change into the directory
```bash
cd master_thesis
```

### 3. Install the source package
```bash
pip install -e .
```

### 4. Explore as you wish
```bash
# For example run main
python ./scripts/main.py
```
